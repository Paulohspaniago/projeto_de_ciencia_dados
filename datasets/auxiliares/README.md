# auxiliares

Pasta para tabelas auxiliares usadas em cruzamentos e enriquecimento.

Uso recomendado neste projeto:

- tabela de municipios do IBGE com `codigo_municipio` de 7 digitos
- equivalencias de nomes de municipios
- tabelas de apoio para padronizacao

Para o notebook de IDH, a estrutura esperada do arquivo `codigos_municipios_ibge.csv` e:

```text
codigo_municipio,codigo_uf_ibge,sigla_uf,nome_municipio,nome_municipio_padronizado
1100015,11,RO,Alta Floresta D'Oeste,alta floresta d'oeste
```
